Hra - Nejlepší had
-------------------------
Toto je nejlepší had ze všech nejlepších hadů
- had začíná v poli 5x5 a postupně se přidává více a více herního pole dokud se nezaplní celý monitor
- varování: Hra je velmi návyková a autor neručí za jakékoliv úrazy a škody na majetku, způsobené hraním této hry.
